jsFantasy

jsFantasy is a MMORPG (Massively Multiplayer Online Role Playing Game) that teaches players JavaScript with a story line. It incorporates lessons as the story moves along. It provides a fun way to learn JavaScript.

Current features:
  • Persistent database and functional login, cannot email you your password if you forget it, so remember it!
  • Command console accepts 4 commands for movement: north(); south(); east(); west();
  • Multiplayer functionality (Right now you can just see the other players, no collision with them yet)
  • Movement between maps
  • Online Chat using command console: type “//” before message.
